# shuttle-formula 的使用示例

shuttle-formula 仅提供了基础的公式编辑和计算能力，可以根据需要灵活扩展样式

## 启动项目

```bash
# clone 项目
git clone git@github.com:Mingbing-get/shuttle-formula-example.git
```

```bash
# 进入项目
cd ./shuttle-formula-example
```

```bash
# 安装项目依赖
pnpm i
```

```bash
# 启动项目
pnpm dev
```

访问：http://localhost:5173/ 可查看示例
